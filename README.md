
# 🌍 ImpactAtlas

> AI-powered NGO Data Intelligence Platform developed for the **Google Solution Challenge 2026**.



## 📌 About

ImpactAtlas is an AI-powered platform designed to help NGOs manage, analyze, and visualize data efficiently. The platform leverages modern web technologies and AI capabilities to improve decision-making and maximize social impact.

This project was developed as part of the **Google Solution Challenge 2026**.

---

## ✨ Features

- 📊 NGO Dashboard
- 🤖 AI-powered Insights
- 📈 Data Visualization
- 🔍 Smart Search
- 📱 Responsive Design
- 🔐 Secure Authentication
